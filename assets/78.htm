你的男友快乐吗?

　　走了这么久，他变了没有？你们之间的恋情是否依然令他感到快乐而幸福，还是变得连交谈都显得困难？何不试试从下面这个测试中寻找答案呢？一个恋爱中的男人也许不会在大街上蹦蹦跳跳，或者哼着小曲留连于雨中，但是只要稍加留意就能发现不少泄露真情的迹象表明你的男友正如此快乐地沉醉于爱河里――一个十分真诚的微笑会令他"电光四射"，一句"亲爱的，我在家"的耳边亲吻能够溶化他的铮铮铁骨。为帮助你明了你们此刻的恋爱关系正处于哪一级状态，我们设计了下面这个小小的问卷。你只需对每一个问题回答"是"或者"不是"，并且累计你的回答情况，你便会发现他是否依然与你同处在快乐的恋爱高速路上。

1．当他与你在饭店约会或者动机场接你时，他会早到吗？
是 否  　　

2．在同他的老板、父母或者客户共进午餐时，他是否至少曾有一次将他的手放在桌下而搭在你的大腿上。
是 否 

3．你们一起共度良宵后的第二天早上，他会凝视着你就好像他以前从未见过这样美丽的女人吗？
是 否 

4．当你给他一个工作上的建议，他会听吗？
是 否 

5．他能准确地解释出你工作的性质并且叫出你们单位的几位关键人物的名字吗？
是 否 

6．某个星期日的7：00，闹钟铃响后，在是否有时会向你表达这样的意思：亲爱的，我真很后悔答应别人今天一起打网球，去登山或者和兄弟们一块玩高尔夫，因为我更宁愿整天都和你呆在床上，听听音乐并且呷两口小酒？
是 否 

7．他连着两天都去打网球，爬山或者玩高尔夫，却没有带上你，但他仍然会找一个适当的时候打电话告诉你，他正想着你吗？
是 否 

8．在他的钱包或者每日记事簿里是否夹有你的照片呢？
是 否 

9．他是否曾在一个陌生人面前吹嘘过你？
是 否 

10． 在他失眠的时候，他有时会要求蜷曲在你的臂弯以帮助他人睡吗？
是 否 

11． 当你结束一次旅行回家时，他会用一脸甜蜜而充满爱意的表情久久地看着你吗？
是 否 

12． 当他向你发誓哪怕你素面朝天也一样十分漂亮时，他的话令你信服吗？
是 否 

13． 他会及时解除你的某些疑虑吗？例如，当你抱着沉沉的三大包食品气喘吁吁地爬楼梯时，他却在愉快地分捡信件――但是当他意识到他刚才是多么粗心大意，考虑不周时，他又会及时向你道歉。
是 否 

14． 他愿意将你介绍给他的异性朋友认识？
是 否 

15． 他是否放弃过一次很棒的外出旅游的机会只因为不能带你同行？
是 否 

16． 他喜欢坐在床上并且看着你换衣服吗？
是 否 

17． 他会让你穿他的衬衫――最好的那几件吗？
是 否 

18． 在公园里看着那些聪明伶俐的小孩玩耍时，他的目光是否曾经显得朦胧而柔和，然后还充满暖意地看着你？
是 否 

19． 在同你一道看电视时，他会故意跳过他所喜爱的体育频道吗？
是 否 

20． 和你一同去拜望你最亲近的一位阿姨时，他是讨人喜欢的人吗？
是 否 

21． 每个星期至少有一次你会令他开怀大笑吗？
是 否 

22． 在床上，他会让你把冰凉的双脚放在他的两腿之间吗？
是 否 

23． 他有时候会为你做饭吗――或者至少试图这样做。
是 否 

24． 他让你快乐吗？
是 否
