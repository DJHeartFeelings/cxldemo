var node = new Ext.tree.TreeNode({text:"节点名",hobby:"跳伞"}),
使用node.attributes.hobby既可以取出hobby的值。按这种方法可以添加任何需要的属性到节点中。

每一个treenode都有一个TreeNodeUI和它对应


TreePanel 类常用方法：
getNodeById( String id ) : Node 返回节点
getRootNode() : Node 返回树的根节点 

getSelectionModel() : TreeSelectionModel 返回这个treePanel所用的选择模型 ,
默认为DefaultSelectionModel,只支持单选，要支持多选，改为MultiSelectionModel，
在TreePanel配置 selModel:new Ext.tree.MultiSelectionModel()

带复选框的节点，在TreeNode配置checked：false/true，TreePanel的getChecked()方法返回所有处于选中状态的节点，
getChecked("id")返回所有id，getChecked("text")返回所有节点文本

Ext.tree.AsyncTreeNode 异步树节点，通过配置loader : new Ext.tree.TreeLoader或dataUrl进行加载，每次点击节点，就会发送请求加载，传递node节点的id过去。
返回信息格式： [{id:1,text'nodetext',leaf:'true'}...............]
